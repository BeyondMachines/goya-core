# main
The main app of the project. 
It contains the public pages/content interface and the central logic of managing the registered workspaces - types of notifications, frequency, categories etc.  
The UI is Bootrap 5 based.

## Endpoints defined in this app
- `home` - the endpoint that creates the home page of the Goya web application.

## Models defined in this app
- `SlackInstalledWorkspace` - The model that stores the register of workspaces where the App is successfully installed.
